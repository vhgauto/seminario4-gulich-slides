# Better science communication with `shiny` apps

Repositorio de la presentación del Seminario #4 "Better science communication with `shiny` apps" del MS Víctor Gauto, para el Insituto Gulich.

# [Link a la presentación](https://vhgauto.github.io/seminario4-gulich-slides/)

La forma tradicional de comunicar ciencia, mediante publicaciones y pósters, coloca siempre a nuestra audiencia en una postura pasiva frente a nuestros resultados. Las aplicaciones permiten una participación activa de nuestros lectores, permitiéndoles la posibilidad de explorar y manipular los datos, las visualizaciones y las tablas. shiny es una herramienta, con soporte para R y Python, para la construcción de aplicaciones de forma fácil y basadas en datos. Las posibilidades de shiny serán exploradas mediante un ejemplo basado en teledetección sobre el Río Paraná.

Aplicación [Proyecto Paraná](https://vhgauto-seminario4-gulich.share.connect.posit.cloud).
